# 📊 Amazon Sales Analytics Project

## 🔹 Project Overview
This project analyzes a large-scale amazon sales dataset (100K+ records) to uncover actionable business insights related to sales performance, customer behavior, and product trends.

The project demonstrates an end-to-end data analysis workflow — from data cleaning to visualization — using SQL, Python, and Power BI.

---

## 🔹 Problem Statement
Businesses often face challenges in:
- Tracking revenue trends over time  
- Identifying top-performing products and categories  
- Understanding customer purchasing behavior  

This project aims to solve these challenges using data-driven insights.

---

## 🔹 Tools & Technologies
- Python (Pandas for data cleaning & preprocessing)  
- MySQL (data querying & analysis)  
- Microsoft Power BI (data visualization & dashboarding)  
- Excel (raw dataset)

---

## 🔹 Dataset Details
- Dataset contains 100,000+ sales records  
- Key fields include:
  - Order ID, Order Date  
  - SKU, Category, Size  
  - Quantity & Revenue  
  - Shipping City, State, Country  
  - Order Status & Fulfilment  

---

## 🔹 Data Cleaning & Preparation
- Handled missing and null values  
- Converted date format from `DD-MM-YYYY` (text) to SQL `DATE` format (`YYYY-MM-DD`)  
- Removed duplicate records  
- Standardized categorical values  
- Validated numeric columns (amount, quantity)

---

## 🔹 SQL Analysis Performed
- Monthly revenue and order trends  
- Top 10 categories by revenue  
- Product-level analysis using SKU  
- State-wise and city-wise order distribution  
- Aggregations using GROUP BY, JOINS, and aggregate functions  

---

## 🔹 Power BI Dashboard
The interactive dashboard includes:
- Revenue trend (line chart)  
- Category-wise performance (bar chart)  
- Geographic sales distribution (map)  
- KPI Cards:
  - Total Revenue  
  - Total Orders  
  - Total Quantity  

---

## 🔹 Key Insights
- Identified peak sales months and seasonal trends  
- Found top-performing categories contributing maximum revenue  
- Highlighted regions generating the highest number of orders  
- Analyzed customer purchase patterns  

---

## 🔹 Project Workflow
→ Excel (Raw Data)  
→ Python (Data Cleaning)  
→ MySQL (Data Analysis)  
→ Power BI (Dashboard & Insights)

---

## 🔹 How to Run the Project
1. Import dataset into MySQL  
2. Perform data cleaning using Python/SQL  
3. Run SQL queries for analysis  
4. Connect Power BI to MySQL database  
5. Refresh dashboard to view insights
